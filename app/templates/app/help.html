Image Inference

This section explains how to use the image inference feature of our web application.

1. Select the "Run Analysis" button on the Home page to run inference for a disaster.

2. Please complete all mandatory fields and ensure that the file extension for both pre-disaster and post-disaster images is either .tiff or .tif.

3. Submit all the information and await the inference results.

4. Once the images have been processed, you will be directed to the dashboard. To review past inferences, navigate to the profile page.

Dashboard

This section describes the how to navigate the dashboard functionality of our web application.

1. Following the completion of inference, User is directed to the dshboard.

2. At the top of the page, demographic and weather-related information about the damaged area is available.

3. Statistical information is displayed in the form of KPIs, charts, lists, and map. To get count and percentages, hover over the charts.

4. Navigate to the interactive map at the bottom-right conner for detailed map visualization..

Map Visualization

This section provides information on how to navigate the map visualization in our web application.

1. Following the completion of inference, a compact map appears in the bottom-right corner of the dashboard, displaying a heatmap overlay of the region affected by damage.

2. To get a more detailed view, click the interactive map icon.

3. On the interactive map page, you can access detailed maps pinpointing the locations of damage.

4. Toggle buttons at the map's bottom offer various options to display distinct damage classes.

5. You can adjust the map display between streetmap and satellite views by selecting the options in the top-right corner of the map.
